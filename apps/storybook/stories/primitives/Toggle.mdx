Single pressable button that holds two states (off / on) built on
`@radix-ui/react-toggle`. Common in composer toolbars (Bold, Italic), inline
filters, and "pin / star" affordances. For mutually exclusive choices in a
row, use **ToggleGroup**.

## States

| State | Visual |
| --- | --- |
| default (off) | `bg-transparent` + `text-fg-default` + transparent border |
| hover (off) | `bg-state-hover` overlay |
| **pressed (on)** | `bg-accent-bg-subtle` + `text-accent-text` + `border-accent-border` |
| disabled (any) | `opacity-50` + `cursor-not-allowed` |
| focus | Global `:focus-visible` ring (Toggle is a `<button>`) |

The pressed state matches the filter-chip pattern from States.md — two
signals together (color + border), never color alone (WCAG 1.4.1).

## Sizes

Same scale as Button — `sm` / `md` / `lg` for label toggles, `icon-sm` /
`icon-md` / `icon-lg` for icon-only (composer toolbar default).

## Tokens used

| Token | Where |
| --- | --- |
| `--color-state-hover` / `-state-active` | Hover / active overlay (off state) |
| `--color-accent-bg-subtle` | Pressed background |
| `--color-accent-text` | Pressed text |
| `--color-accent-border` | Pressed border |
| `--color-fg-default` | Resting label |
| `--text-label-sm` / `-label-md` | Type by size |
| `--radius-md` | Corner |
| `--duration-fast` + `--ease-standard` | Transitions |

## A11y

- Renders `role="button"` with `aria-pressed` synced by Radix (`true` / `false`).
- Icon-only toggles **require `aria-label`** — the IconButton convention applies (the stories above pass labels).
- For state-bearing toggles (Bold / Italic), the label should reflect *what gets toggled*, not the current state — e.g. `aria-label="Bold"`, not `aria-label="Bold off"`.

## Forbidden

- Toggle for a *command* — use `Button`. Toggles persist state; Buttons fire actions.
- Toggle for *one-of-many* in a row — use `ToggleGroup type="single"` (single tab-stop, arrow-key nav)
- Inline `aria-pressed` management — let Radix do it via `pressed` / `defaultPressed` props
- A `size` of `xs` / `xl` — locked to the standard control scale
