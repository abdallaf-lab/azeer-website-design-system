## Item highlight

States.md, DropdownMenu / ContextMenu item row:

> Hover Б†’ bg `accent-bg-subtle` + text `accent-text` (= same as selected) Бқ” menu items use hover-style for keyboard nav.

So **mouse hover and keyboard focus produce the same highlight**. Radix sets
`data-[highlighted]` on the active item; our CSS targets that attribute:

```tsx
"data-[highlighted]:bg-accent-bg-subtle data-[highlighted]:text-accent-text"
```

Destructive items invert the intent Бқ” danger background tint + danger text:

```tsx
"data-[highlighted]:bg-danger-bg-subtle data-[highlighted]:text-danger-text"
```

## Keyboard

Radix wires the full menu pattern:

| Key | Behavior |
| --- | --- |
| `Б†‘` / `Б†“` | Move highlight |
| `Enter` / `Space` | Activate highlighted item |
| `Esc` | Close menu, return focus to trigger |
| `Home` / `End` | Jump to first / last item |
| `A`-`Z` | Type-ahead to first matching item |
| `Tab` | Closes menu (matches WAI-ARIA menu pattern) |

## A11y

- Trigger needs to be focusable Бқ” wrap `Button` with `asChild`.
- Each item is `role="menuitem"` (Radix). `aria-disabled` set automatically for `disabled` items.
- Group labels render `<DropdownMenu.Label>` which carries `role="presentation"` Бқ” purely visual.
- Destructive items don't get a separate `aria-` annotation Бқ” color + icon + verb together signal intent. Pair with a `ConfirmDialog` for the actual irreversible step.

## Forbidden

- Custom hover colors per-menu Бқ” `accent-bg-subtle` highlight, period
- Menu items without an icon when other items have icons Бқ” pad with empty space via `inset` instead, never mismatched padding
- Two destructive items in one menu Бқ” there's at most one irreversible "delete-this" action per context
- Submenu (`DropdownMenuSub`) Бқ” deferred to a later batch (you can add via Radix directly if urgent)
- Checkbox/Radio items (`CheckboxItem` / `RadioItem`) Бқ” deferred; use a Popover with form controls if needed today
- Custom shortcut formatting Бқ” wrap real `<Kbd>` inside `DropdownMenuShortcut`, never raw text
- DropdownMenu for an action that needs >7 items Бқ” switch to a Combobox / Command palette
