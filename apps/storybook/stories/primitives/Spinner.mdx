## Anatomy

- Wraps Lucide's `Loader2` icon inside a `<span role="status" aria-live="polite">`.
- The icon is `aria-hidden="true"` — the live region announces via the visually-hidden `sr-only` label (default `"Loading"`).
- Color inherits `currentColor` from the parent — wrap in a colored context to tint.

## Sizes

| Size | Pixels |
| --- | --- |
| `sm` | 16 |
| `md` (default) | 20 |
| `lg` | 24 |

These match the DS icon scale (Foundation §6) — pass `sm` / `md` / `lg`, never
arbitrary pixel sizes.

## Tokens used

| Token | Where |
| --- | --- |
| `--color-fg-muted` (`text-fg-muted`) | Default color (inherited via `currentColor`) |
| `animate-spin` (Tailwind built-in) | Rotation keyframe |

Stroke width is locked to 1.5 in the Icon wrapper — Spinner inherits.

## States

Single state — animating. There is no "stopped" or "complete" Spinner; mount
when work starts, unmount when it finishes.

## A11y

- `role="status"` + `aria-live="polite"` so screen readers announce when the spinner appears.
- The Lucide icon is `aria-hidden`; the `sr-only` label inside provides the accessible name.
- `prefers-reduced-motion`: the global rule in `tokens.css` disables all CSS keyframe animations, so the spinner stops moving — readers still announce the live region.

## Forbidden

- Spinner that runs > 2 seconds — switch to `Skeleton` (containers) or `Progress` (measurable)
- Multiple spinners for the same action
- Spinner stacked with Skeleton on the same surface — pick one
- Spinner replacing button text without preserving the button's width — use `Button` `loading` prop (Batch 2)
- Custom spin animations or durations — use `animate-spin` (Tailwind built-in)
- Custom colors per feature — inherit `currentColor`
- Arbitrary pixel sizes — only `sm` / `md` / `lg`
