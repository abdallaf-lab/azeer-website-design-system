# Azeer Brand — دليل هوية أزير

The marketing-brand surface of Azeer, structured from the official brand book.
This **Brand** section is distinct from the **Foundation** section: Foundation
documents the *product-UI token system* (the app), while Brand documents the
*marketing identity* (logo, brand palette, voice, patterns) used on the website,
decks, social, and print.

<Callout intent="info" title="Two layers, one system">
Brand colors live as `--brand-*` vars and `bg-brand-*` utilities (see **Brand →
Color**). The product UI re-derives an accessible indigo family from them — e.g.
brand primary `#7B61FF` is a non-text identity color, while the product uses
`--color-accent-fill` `#5238D1` for white-text CTAs. Use brand tokens for
marketing surfaces; use the semantic `--color-*` tokens for app UI.
</Callout>

<Callout intent="ai" title="Consume it in code">
Everything here is typed and importable. Brand metadata: `import { BRAND_COLORS,
BRAND_VOICE, BRAND_WRITING_RULES, … } from "@azeer/tokens"`. Brand components:
`import { AzeerLogo, AzeerMark, BrandPattern } from "@azeer/website-ui"`. See
**Brand → Usage in Code**.
</Callout>
