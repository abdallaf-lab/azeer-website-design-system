# Patterns — النمط

Four brand patterns, each derived from the desert + the mark. Rendered live from
`<BrandPattern>` — all draw in `currentColor`, so a `text-*` utility tints them.

<PatternGallery />

<Callout intent="info" title="Using a pattern as a background">
Place it absolutely behind content in a positioned, clipped wrapper and tone it
down with opacity:

```tsx
<div className="relative isolate overflow-hidden bg-brand-supportive">
  <BrandPattern
    variant="waves"
    className="absolute inset-0 text-brand-secondary opacity-10"
  />
  <div className="relative">…content…</div>
</div>
```
</Callout>

<Callout intent="warning" title="Don't put the logo on a pattern">
Patterns are background texture only. Per the logo rules, never place the logo
over a patterned background — keep brand surfaces legible and high-contrast.
</Callout>

In code: `import { BRAND_PATTERNS } from "@azeer/tokens"` lists each variant with
its inspiration and meaning; `import { BrandPattern } from "@azeer/website-ui"`
renders them.
