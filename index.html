Single source of truth for designing, building, and auditing any Azeer screen, component, or flow. If a value is not in this system, ask — don't invent. Every color binds to a token, every space is on the 4-pt grid, every component declares which Surface Profile it belongs to.

Surface Profiles

Consumer Booking

Patients & shoppers. 2–5 minute task flows. Tenant brand owns the color; Azeer is footer attribution only. Light-only.

  • Confirmation

AI-PROMPT-BLOCK.md

Non-negotiable rules

01

Every value is a token

No raw hex, no off-scale spacing, no one-off font sizes. If it's not in the system, ask.

02

Profile awareness

Never mix Profile A density into Profile B. Never render tenant theming in Profile A.

03

Seven states

Default, hover, focus, disabled, loading, empty, error. Every interactive component.

04

RTL is mandatory

Logical properties only. Mirror layout, never data — phone numbers & IDs stay LTR.
